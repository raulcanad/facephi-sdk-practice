

## Configuración y Ejecución con VS Code Live Server

### **Método Recomendado: VS Code + Live Server**

1. **Instalar Visual Studio Code:**
   - Descarga desde [code.visualstudio.com](https://code.visualstudio.com/)
   - Instala en tu sistema

2. **Instalar Extensión Live Server:**
   - Abre VS Code
   - Ve a **Extensions** (Ctrl+Shift+X)
   - Busca **"Live Server"** de Ritwick Dey
   - Haz clic en **Install**

3. **Abrir el Proyecto:**
   ```bash
   cd facephi_practice
   code .

## Cómo Usar
1. Abre `index.html` en cualquier navegador moderno
2. Explora las 4 secciones principales:
   - **Verificación Facial**: Activa cámara y captura selfies
   - **Verificación Documento**: Sube y valida documentos
   - **Llamadas API**: Simula comunicación con backend
   - **Consola Debug**: Monitorea logs y errores

3. Prueba diferentes escenarios:
   - Flujo normal (happy path)
   - Errores de permisos
   - Fallos de API
   - Validaciones rechazadas
